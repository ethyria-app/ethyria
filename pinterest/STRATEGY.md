## 2. Pin-Erstellung

### Dateien in diesem Ordner

| Datei                  | Zweck                                                                     |
| ---------------------- | ------------------------------------------------------------------------- |
| `pin-template-de.html` | Visuelles Pin-Design DE — im Browser öffnen & screenshotten (1000×1500px) |
| `pin-template-en.html` | Visuelles Pin-Design EN — im Browser öffnen & screenshotten (1000×1500px) |
| `pin-texts.md`         | Fertige Titel, Beschreibungen & Hashtags (Copy & Paste)                   |

### So erstellst du den Pin:

1. Öffne `pin-template-de.html` im Browser
2. Screenshot machen (exakt 1000×1500px) oder mit Browser DevTools auf 1000×1500 setzen
3. Alternativ: Pin in Canva nachbauen mit den gleichen Farben/Texten
4. Pin auf Pinterest hochladen
5. Titel & Beschreibung aus `pin-texts.md` kopieren

### Design-Spezifikationen

- **Format:** 1000 × 1500px (Pinterest optimal: 2:3)
- **Hintergrund:** Gradient #100c1f → #08134e → #001a79
- **Akzentfarbe:** #01bfff (Cyan) / #0066ff (Blau)
- **Headline-Font:** Poppins Bold/ExtraBold
- **Body-Font:** Inter Regular/SemiBold
- **Textfarbe:** #ffffff (Headlines), #d9e4ff (Body), #91a0c6 (Subtle)

## 3. Posting-Strategie

### Frequenz

- **Woche 1–2:** 1 Pin pro Tag (verschiedene Varianten)
- **Ab Woche 3:** 3–5 Pins pro Woche
- **Content-Mix:** 60% Ethyria-Pins, 40% allgemeine Traumdeutungs-Inhalte

### Beste Posting-Zeiten (DACH + International)

| Zeit (MEZ)    | Zielgruppe          |
| ------------- | ------------------- |
| 08:00 – 09:00 | Morgens (DE/AT/CH)  |
| 12:00 – 13:00 | Mittagspause        |
| 20:00 – 22:00 | Abends / Prime Time |

### Pin-Varianten rotieren

1. **Haupt-Pin** — "Deine Träume haben eine Bedeutung"
2. **Emotionaler Hook** — "Letzte Nacht geträumt?"
3. **Feature-Fokus** — "Dein Traumtagebuch, das mitdenkt"
4. **Minimal/Neugier** — "Freud & Jung in deiner Tasche"

→ Pro Variante 2–3 visuelle Versionen (verschiedene Screenshots, Farb-Akzente)
