# SEO Keyword Map

This file turns the landing page SEO strategy into concrete keyword targets by language.

## Positioning Thesis

Ethyria should not compete as only a generic dream journal or only a generic AI app.
The strongest position is a hybrid:

- AI dream interpretation app
- private dream journal app
- psychology-based dream analysis app
- Android dream diary with visual insights
